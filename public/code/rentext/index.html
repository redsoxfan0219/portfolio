RenText

RenText is my ongoing effort to use Python and advanced analytics to analyze Renaissance English texts. The data used in this project is from the Early English Books Online (EEBO) Text Creation Partnership.

  • Site built using the docs as code approach via Git & Hugo. Source materials are stored on GitHub and served via continuous deployment to Netlify.
